About Stock Predictor

This Stock Predictor is an advanced tool crafted to analyze and visualize stock market trends, with a specific focus on Safaricom (SCOM) stock. This project was developed as an academic initiative aimed at exploring financial market analysis and visualization techniques.

Features

  • Visualization of historical stock data
  • Computation of moving averages (100-day and 200-day)
  • Statistical evaluation of stock performance
  • Interactive charts for enhanced trend analysis

Technology Stack

  • Python: The primary programming language used for backend processing and data analysis
  • Django: A robust Python framework for efficient web development
  • Pandas: A powerful library for data analysis and manipulation
  • Matplotlib: A versatile visualization library for generating static, animated, and interactive plots
  • HTML/CSS: Utilized for page structure and styling

Developer

Leah Sigana developed this project as part of a personal academic pursuit. It demonstrates practical applications of data analysis and web development in stock market forecasting.
